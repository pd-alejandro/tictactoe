# Tic-Tac-Toe Game

A modern, fast, and accessible Tic-Tac-Toe game built on the web. 

## Features
* **Built with React & TypeScript** for reliable and clean code.
* **Modern CSS Styling**
* **Celebration effects** using `canvas-confetti` when a player wins.

## Tech Stack
* **Framework:** React
* **Language:** TypeScript
* **Styling:** CSS Modules
* **Building:** Vite
